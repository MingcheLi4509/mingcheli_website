# Ming-Che Li Personal Website

A clean static personal website for Ming-Che Li, IC designer.

## Local Preview

Open `index.html` directly in a browser, or run a simple local server:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Edit Later

- `index.html` contains the content.
- `styles.css` contains the visual design.
- Replace placeholder links, projects, proof metrics, and resume link when ready.

## GitHub Pages

This site is ready for GitHub Pages. Push these files to a GitHub repository and enable Pages from the repository settings.
